# Kidney Disease Prediction using Machine Learning

## Project Overview
This project predicts whether a patient has kidney disease or not using Machine Learning. 
The model is built using the Random Forest Classification algorithm.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Machine Learning Algorithm
- Random Forest Classifier

## Dataset
The dataset contains medical attributes such as blood pressure, sugar level, hemoglobin, etc., which are used to predict kidney disease.

## Project Steps
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Selection
5. Model Training using Random Forest
6. Model Evaluation

## How to Run the Project
1. Download the project files
2. Open the notebook in Jupyter Notebook
3. Run all cells to train the model
